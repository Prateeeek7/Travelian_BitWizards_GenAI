### Project Overview

**Travelian India** is a modern, AI-powered travel planning platform specifically designed for Indian travelers. Built with React + FastAPI architecture, it leverages advanced AI agents to create personalized, budget-aware travel itineraries with beautiful UI and interactive maps.
